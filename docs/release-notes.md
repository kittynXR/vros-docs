# Release Notes

## April 20, 2026

The public docs surface was reset on **April 20, 2026** to prepare for the final vrOS release.

### What changed

- The docs center was narrowed to release-facing material only.
- Install, first-run, control center, overlays, integrations, troubleshooting, FAQ, and support routing were prioritized.
- Older research-heavy sections were removed from public navigation during release prep.

### Support links

- Main site: [vros.cat](https://vros.cat)
- Docs: [docs.vros.cat](https://docs.vros.cat)
- Support: [support.vros.cat](https://support.vros.cat)

More public release notes can be added here as launch builds ship.
