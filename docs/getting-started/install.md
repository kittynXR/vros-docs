# Install vrOS

vrOS is distributed through Steam. The release path is tuned for Windows first, with SteamVR as the primary VR runtime.

## Before you install

- Windows 11 is the primary target for the release build.
- SteamVR should already be installed and able to launch cleanly on its own.
- A SteamVR-compatible headset is required for the VR overlay experience.
- A stable GPU driver and enough headroom for capture overlays matter more than raw overlay count.

## Install steps

1. Open the [vrOS Steam page](https://store.steampowered.com/app/3873610).
2. Install vrOS from your Steam library.
3. Launch vrOS once from Steam so the desktop control center and runtime can initialize.
4. Start SteamVR after the first launch if it was not already running.
5. Confirm both the desktop control center and the SteamVR dashboard entry appear.

## Verify the install

- The desktop control center opens on Windows.
- SteamVR shows the vrOS entry in the dashboard or startup surfaces.
- A basic overlay, such as a dashboard or capture surface, can be opened without restarting the whole PC.

## First-install caveat

If vrOS installs successfully but no overlay appears in SteamVR, do one full SteamVR restart before changing settings. That is the most common first-run recovery step.
