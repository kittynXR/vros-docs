# Integrations

The release-facing integration story centers on creator workflows that benefit from living close to the overlay stack.

## Launch integrations

### OBS

Use vrOS to keep scene, source, stream, or recording controls near your VR workflow. Launch OBS first, then connect it from the control center.

### Twitch

Chat and creator workflows can be attached to the same environment as your overlays so you are not constantly dropping back to the desktop.

### VRChat OSC

Use the VRChat OSC path for connected workflows that need to live alongside the rest of your creator tooling.

## Integration troubleshooting

- Reconnect the integration before assuming the saved session is still valid.
- Confirm local services are actually running.
- Check firewall or local network rules if a connection works one day and silently fails the next.
- Include the exact integration and action sequence when filing a bug.
